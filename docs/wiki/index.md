# 숭산텍(SIT) 지식 위키

환영합니다! 숭산텍(SoongSanTech) 지식 위키는 자율주행 연구 프로젝트를 진행하며 축적된 이론, 기술 스택, 시스템 아키텍처 문서를 체계적으로 정리하는 공간입니다.

## 📌 주요 카테고리

<div class="grid cards" markdown>

-   :material-rocket-launch:{ .lg .middle } __시작하기__

    ---

    프로젝트의 전반적인 개요, 팀 구성, 그리고 개발 환경 셋업 방법을 안내합니다.

    [프로젝트 개요 보기](getting-started/project-overview.md)

-   :material-school:{ .lg .middle } __이론 (Theory)__

    ---

    자율주행의 기초 개념부터 행동복제(BC), 강화학습(PPO), Sim-to-Real 전이 등 핵심 연구 이론을 다룹니다.

    [이론 문서 보기](theory/autonomous-driving-overview.md)

-   :material-toolbox:{ .lg .middle } __기술 스택 (Tech Stack)__

    ---

    CARLA 시뮬레이터, PyTorch, ROS2, TensorRT 등 프로젝트에 사용되는 주요 기술의 가이드와 팁을 제공합니다.

    [기술 스택 보기](tech-stack/carla-simulator.md)

-   :material-graph:{ .lg .middle } __아키텍처 (Architecture)__

    ---

    데이터 수집 파이프라인, 모델 학습 루프, 엣지 디바이스(Jetson) 배포 아키텍처 등 시스템 설계 문서를 모아두었습니다.

    [아키텍처 문서 보기](architecture/data-pipeline.md)

</div>

## 🔍 문서 검색 및 활용

우측 상단의 **검색창(Search)**을 통해 필요한 문서를 빠르게 찾을 수 있습니다. 모든 문서는 마크다운 기반으로 작성되어 있으며, 코드 블록, 수식(MathJax), 다이어그램(Mermaid)을 적극 활용하여 가독성을 높였습니다.

새로운 지식을 추가하거나 기존 문서를 수정하고 싶다면, [기여하기 가이드](../contributing/index.md)를 참고해 주세요.
