# 이론부터 실증까지,<br>모든 기술적 맥락을 공유합니다.

<p class="hero-lead">
자율주행, 강화학습, CARLA 시뮬레이터, ROS2 등 숭산텍 프로젝트의 근간이 되는 핵심 개념과 시스템 아키텍처, 기술 스택 가이드를 체계적으로 정리합니다.

## 카테고리

<div class="grid cards" markdown>

- :material-book-open-page-variant-outline:{ .lg .middle } **이론 (Theory)**

    ---

    자율주행의 기본 개념부터 행동복제(BC), PPO 강화학습, Sim-to-Real 전이 등 프로젝트를 지탱하는 학술적·이론적 배경을 다룹니다.

- :material-layers-triple-outline:{ .lg .middle } **기술 스택 (Tech Stack)**

    ---

    CARLA, PyTorch, TensorRT, ROS2 등 숭산텍이 채택한 핵심 기술들의 사용법, 최적화 기법, 트러블슈팅 가이드를 공유합니다.

- :material-sitemap-outline:{ .lg .middle } **시스템 아키텍처 (Architecture)**

    ---

    데이터 수집 파이프라인, 모델 학습 워크플로우, 엣지 디바이스 배포 구조 등 전체 시스템의 큰 그림과 컴포넌트 간 상호작용을 설명합니다.

- :material-rocket-launch-outline:{ .lg .middle } **시작하기 (Getting Started)**

    ---

    프로젝트의 전반적인 목표와 방향성, 팀 구성, 그리고 새로운 기여자를 위한 로컬 개발 환경 셋업 가이드를 제공합니다.

</div>
