# 팀 구성 및 자원

숭산텍(SIT)은 자율주행 연구·실증에 특화된 소규모 팀으로, 고가의 실차나 대규모 인프라 없이 시뮬레이터와 RC카를 활용하는 **저비용 고효율 연구 환경**을 구성하고 있습니다.

## 1. 팀 자원 요약

| 항목 | 내용 |
|------|------|
| 팀 규모 | 2명 (확장 가능) |
| 컴퓨팅 | NVIDIA RTX 4090 × 1, RTX 3090 Ti × 1 |
| 시뮬레이터 | CARLA 0.9.15 (Windows Host + WSL2 Client) |
| 엣지 디바이스 | NVIDIA Jetson Xavier NX / Orin Nano |
| 배포 플랫폼 | 1/10 스케일 RC카 |
| 미들웨어 | ROS2 Humble |

## 2. 협업 도구

| 도구 | 용도 |
|------|------|
| GitHub | 코드·문서 버전 관리, 이슈 트래킹, PR 리뷰 |
| GitHub Pages + MkDocs | 본 위키 및 R&D 로깅 사이트 |
| Discord / Slack | 일상 커뮤니케이션 (선택) |
| Notion / Linear | 태스크 관리 (선택) |

## 3. 외부 협력 및 공개

본 프로젝트는 모든 코드와 문서를 GitHub를 통해 공개하며, 자율주행 연구 커뮤니티에 기여하는 것을 지향합니다.
