# 검증된 것들을 잘 조합하여,<br>의미 있는 결과를 만듭니다.

<p class="hero-lead">
SoongSanTech (SIT, 숭산텍)는 인공지능을 활용한 응용 서비스 개발에 주력하는 소규모 R&D 팀입니다. 현재는 디지털 트윈 환경(CARLA)에서 자율주행 모델을 개발하고, Sim-to-Real 전이가 가능한 경량 모델을 연구합니다.
</p>

## 우리의 신념

> "새로운 것을 발명하기보다는, 검증된 것들을 잘 조합하여 의미 있는 결과를 만든다."

자원이 제한된 소규모 팀이지만, 명확한 방향성과 체계적인 실증을 통해 학계와 산업계 모두에 가치 있는 기여를 하고자 합니다. 모든 연구 과정과 실패의 흔적을 투명하게 공개하는 것이 우리의 차별점입니다.

## 운영 원칙

<div class="grid cards" markdown>

- :material-eye-outline:{ .lg .middle } **Transparency**

    ---

    모든 코드와 연구 과정을 공개합니다. 이 사이트와 GitHub 레포지토리가 그 대표적 예입니다. 실패와 시행착오까지 가감 없이 기록합니다.

- :material-refresh:{ .lg .middle } **Reproducibility**

    ---

    실험은 누구나 동일한 결과를 재현할 수 있도록 환경·시드·하이퍼파라미터·데이터셋 버전까지 명시합니다.

- :material-account-group-outline:{ .lg .middle } **Collaboration**

    ---

    2인 이상의 팀 구성을 전제로, 표준화된 템플릿과 PR 기반 워크플로우로 비동기 협업을 추구합니다.

- :material-school-outline:{ .lg .middle } **Continuous Learning**

    ---

    실패도 학습의 일부로 기록하고, 주기적인 회고(Retrospective)를 통해 다음 단계의 의사결정을 개선합니다.

</div>
